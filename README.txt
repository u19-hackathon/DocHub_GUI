# DocHub - интерфейс

Vue.js приложение для управления документооборотом.

## Установка и запуск

### Предварительные требования
- Node.js (версия 14 или выше) - [скачать здесь](https://nodejs.org/)

### Установка

1. Установите Vue CLI глобально (если еще не установлен):
```bash
npm install -g @vue/cli

2. Установите зависимости
```bash
npm install

Примечание: Файл package-lock.json гарантирует установку точных версий зависимостей

3. Запуск
```bash
npm run serve
